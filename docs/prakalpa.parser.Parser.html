<private> _addToken(parserState, currentParseTreeNode)

Adds a token to the concrete syntax tree using a table driven (followSet + stack) top-down predictive parsing algorithm. The grammar must be LL(1) for this to work and Python's grammar is by choice an LL(1) grammar.
